### 调整项2：隐私协议弹窗文案优化

**调整前**：
```
本小程序需要收集省份、缴费年限、平均指数等信息用于计算养老金。
信息仅本地保存，不会上传服务器。

请阅读并同意《隐私保护指引》。
```

**调整后天**：
```
我们需要以下信息来帮您计算养老金：
• 参保地区（用于选择正确计发基数）
• 个人信息（出生年月、工作年限等）
• 缴费情况（缴费指数、账户余额等）

所有信息仅保存在您的手机上，不会上传到任何服务器。
您可以随时清除数据。
```

**调整原因**：
原本文案太技术化，普通用户看不懂。调整后更用户友好，清楚说明需要什么信息、为什么需要、如何保护。

**影响范围**：
- `miniprogram/pages/index/index.wxml`（第8行）
